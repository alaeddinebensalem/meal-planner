# meal-planner
Python application using PostgreSQL to manage recipes, ingredients, and generate optimized shopping lists by store location.

## Features
- Add and manage recipes, ingredients, and store locations
- Generate shopping lists based on selected recipes
- Handles units, purchase amounts, and location-wise grouping

## Tech Stack
- Python 3
- PostgreSQL (psycopg2)
- dotenv for environment variables

## Setup
1. Clone the repository
2. Create a `.env` file with your DATABASE_URL
3. Install dependencies: `pip install -r requirements.txt`
4. Run `python main.py`
